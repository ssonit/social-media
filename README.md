<h1 align='center'><strong>Social App ⚡</strong></h1>

<p>A social network app cloned from Instagram built with ReactJs, Socket.IO and a lots of other new stuff.</p>

#### **Live Demo:** https://social-media-topaz.vercel.app/

## **Main technologies 📝**

- Front-End
  - ReactJs
  - React-Query
  - TypeScript
  - Tailwind CSS
  - SCSS
- Back-End
  - Node.js
  - MongoDB
  - Socket.IO
  - JSON Web Token (JWT)
  - Cloudinary

## **Features 🚀**

- Authentication
  - Sign in with Credential
  - Register new account
- Chatting
  - Create conversation(s)
  - Realtime chat 1-1
  - Show online users
- CRUD post
- Comment on post
- Send reaction to posts, comments
- Search user
- Follow and unfollow user
- View profile
- Toast message
- Drag and drop photo
- Infinite scroll
